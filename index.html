JobMine Website Scraper
JobMine is a web application developed by the University Of Waterloo to help students with the co-op/internship process. The JobMine Website Scraper is a ruby script that scrapes the interview list on a given JobMine account page and sends alerts through SMS/e-mail when a new interview is scheduled. (The code is under a private repo on my GitHub)
